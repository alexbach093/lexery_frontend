# Documentation Index

This directory is organized by purpose instead of keeping Markdown files in the repository root.

- `archive/` contains historical one-off records such as old PR descriptions and change logs.
- `components/` contains component-level notes that should not live inside `src/`.
- `figma/` contains the Figma extraction workflow docs and generated analysis reports.
- `project/` contains current project setup, architecture, and working context notes.

- [`project/BOOTSTRAP.md`](./project/BOOTSTRAP.md) for initial project setup and tooling.

- [`project/context-info.md`](./project/context-info.md) for ongoing project context.
